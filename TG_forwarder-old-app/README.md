# TG_forwarder
Forwarding TG messages  


Packages to install:

telethon - Telegram Python module, more info: https://docs.telethon.dev/en/stable/
gspread - Google sheets reading module https://docs.gspread.org/en/latest/


1. In order to read Google sheets need to do steps in "For Bots: Using Service Account" part in here: https://docs.gspread.org/en/latest/oauth2.html
2. Create Application for TG in me.telegram.org
